Mp3skull Music Downloader

A command-line utility to search and download mp3's from mp3skull.com

Download as .zip Download as .tar.gz View on GitHub

Is finding music hard for you?

This is a small utility that I developed on my spare time to download songs. Whenever I just wanted to get a song that was in my head, I had to go through several hurdles:

  • Remembering to download the song
  • Searching google for a link
  • Waiting for long download times
  • 404 pages and missing files
  • Prohibitive costs
  • Geo-location barriers to buy Amazon MP3's

Does any of these seem familiar?

That is when I found the website mp3skull.com, where a simple search will find many songs. However, it still suffered from slow page and file download times and missing links. So I started this project, with the aim of creating a command line utility to download files from mp3skull.com.

Features

This script is able to quickly search for songs on mp3skull (via page scraping, since there is no API). It also auto-checks the files and displays only valid links. Furthermore, you can see the bitrate, duration, and file size at a glance. All features have been designed with speed and usability in mind. The file checker will use concurrency to find as many valid links as soon as possible and the nice download prompt is simple and easy to use.

Installation

Right now, the only way to install is to clone the git repository and get dependencies by hand using pip. In the future, nicer setup options will be provided.
